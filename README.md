# shtories


## CLI Arguments
|          Flags          |                          Description                           |
| :---------------------: | :------------------------------------------------------------: |
|     `--help`, `-h`      |                        Prints help page                        |
|     `--list`, `-l`      | Lists friends, stories (seen/unseen), incoming friend requests |
|  `--add-friend` `-af`   |                           Add friend                           |
| `--delete-friend` `-df` |                         Delete Friend                          |
|     `--record` `-r`     |                     Begin Story Recording                      |
